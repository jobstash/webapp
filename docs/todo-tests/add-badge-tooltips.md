# add-badge-tooltips - Test Checklist

## Behaviors to Test

### Badge Tooltips

- [ ] Featured badge shows tooltip "Sponsored job posting" on hover
- [ ] Job for Experts badge shows tooltip "Requires proven web3 experience" on hover
- [ ] Job for Web3 Beginners badge shows tooltip "Great for those new to web3" on hover
- [ ] Urgently Hiring badge shows tooltip "This role needs to be filled quickly" on hover

### Tooltip Behavior

- [ ] Tooltips appear on mouse hover
- [ ] Tooltips disappear when mouse leaves badge
- [ ] Tooltips are positioned correctly (not clipped by card boundaries)
- [ ] Tooltips have appropriate delay before showing

### Badge Styling

- [ ] Featured badge has amber/orange gradient styling
- [ ] Expert badge has violet/purple gradient styling
- [ ] Beginner badge has emerald/green gradient styling
- [ ] Urgently Hiring badge has amber/orange gradient styling
- [ ] Unknown badge types fall back to muted styling

### Accessibility

- [ ] Tooltips are accessible via keyboard focus
- [ ] Tooltip content is readable by screen readers
- [ ] Badge remains clickable/interactive with tooltip wrapper
